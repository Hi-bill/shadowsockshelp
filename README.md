## 📖 项目简介

本项目是一个**纯静态单页面网站**，以教程与知识分享为主，内容涵盖：

- ✅ Windows / Android / iOS / macOS / Linux 全平台 SS 配置教程
- ✅ Chrome & Firefox 浏览器 Proxy SwitchyOmega 插件安装与分流配置
- ✅ 全平台 Shadowsocks 客户端备用下载镜像索引
- ✅ 苹果美区 Apple ID 获取与 iOS 小火箭（Shadowrocket）安装指引
- ✅ 用户真实评价 & 典型使用场景案例（E-E-A-T 优化）
- ✅ 高频问题 FAQ（含 Google 富摘要结构化数据）

## 🛠️ 本地预览

无需任何构建工具，直接用浏览器打开即可：

```bash
# 克隆仓库
git clone https://github.com/Hi-bill/shadowsockshelp.git
cd shadowsockshelp

# 直接打开
open index.html

# 或启动本地服务器（推荐，避免部分浏览器跨域限制）
npx serve .
```

## ⚙️ GitHub Pages 部署

1. 进入仓库 **Settings → Pages**
2. Source 选择 `Deploy from a branch`
3. Branch 选择 `main`，目录选择 `/ (root)`
4. 保存后约 1 分钟即可通过 `https://hi-bill.github.io/shadowsockshelp/` 访问
